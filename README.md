 # Apprenticeship Achievement Rates (England, 2024/25) — LEP & IMD Analysis

This repo contains a small, reproducible Python analysis looking at **apprenticeship achievement rates** (achievements ÷ starts) and how they vary by:

- **Local Enterprise Partnership (LEP)** (regional variation + starts vs achievement rate)
- **IMD deprivation quintile** (national pattern)

It includes:
- Data cleaning scripts that produce tidy datasets
- Figure scripts that save charts to `outputs/figures/`
- Simple modelling scripts (OLS/WLS + Binomial GLM) and robustness checks (London influence)
- Basic data quality tests (pytest)
